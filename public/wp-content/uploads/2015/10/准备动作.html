准备动作：

首先：你需要有一块阿里云盘，也就几十块钱，学生的话9.9。【对于个人博客，建议购买流量包，一个月也就30来块钱】

然后：你需要有那么一点linux操作基础。

最后：你需要有一颗客服困难和享受技术带来的幸福。

阿里云进行基本的优化【非必须】

1：修改默认登录路径。阿里云默认是/root，方便每次操作。修改默认登录设在【/home/cuihuan】

2：修复修改路径之后的问题。修改之后会丢失默认路径很多文件，因为，阿里云将默认的配置文件放在/root下面，所以全部拷贝过去 防止再次登录出现名称问题。 修改之后如下

[cuixiaohuan@cuihuan html]# cd
[cuixiaohuan@cuihuan ~]# pwd
/home/cuixiaohuan

一：基于yum 搭建，做yum更新。

yum update

其他常用命令可以参考：http://blog.csdn.net/phpfengzikai/article/details/7085592

同时，更新全部的信息，用阿里云自带的：http://help.aliyun.com/knowledge_detail.htm?knowledgeId=5974184

注意：即使这样子更新了，也会有很多需要亲自下载，后面会说到

二：lamp 环境搭建

一键安装 apache php mysql

yum -y install httpd php mysql mysql-server php-mysql httpd-manual mod_ssl mod_perl mod_auth_mysql php-mcrypt php-gd php-xml php-mbstring php-ldap php-pear php-xmlrpc mysql-connector-odbc mysql-devel libdbi-dbd-mysql

问题：如果选择的是centos6.0 以上，默认没有mysql，默认是mariadb。所以需要从社区yum中获取最新的mysql， 操作过程如下

1：获取社区的rpm包

rpm -Uvh http://dev.mysql.com/get/mysql-community-release-el7-5.noarch.rpm

[cuixiaohuan@cuihuan ~]# rpm -Uvh http://dev.mysql.com/get/mysql-community-release-el7-5.noarch.rpm
Retrieving http://dev.mysql.com/get/mysql-community-release-el7-5.noarch.rpm
Preparing...                          ################################# [100%]
Updating / installing...
1:mysql-community-release-el7-5    ################################# [100%]
[cuixiaohuan@cuihuan ~]# yum repolist enabled | grep "mysql.*-community.*"

2:查询可用的mysql和mysqlserver

yum search mysql
mysql-connectors-community/x86_64 MySQL Connectors Community                 17
mysql-tools-community/x86_64      MySQL Tools Community                      23
mysql56-community/x86_64          MySQL 5.6 Community Server                184

3：安装

yum -y install mysql-community-server

4：进行基本配置，开机自动启动，启动进程【此处略去】。

三：环境配置和优化

1：apache多端口配置：

apache一个服务器下多个端口承载不同的开发和分布任务，对于初期开发尤为重要

在httpd.conf 增加 以下端口

Listen 1314
<VirtualHost *:1314>
    ServerAdmin cuihuan
    DocumentRoot /var/www/1314/
    #ServerName 1314
    #Alias "/var/www/1314/"
<Directory "/var/www/1314">
    Options All
    AllowOverride All
    Order allow,deny
    allow from all
</Directory>
</VirtualHost>

四：后续开发优化

安装开发中必须的cvs 和数据库管理phpmyadmin

1：安装svn

两个主要目的：1：是作为项目代码控制，2：文件传输服务器

之后所做的事情就是在上面添加svn，参考：http://www.ha97.com/4467.html

启动：

sudo svnserve -d -r /home/cuixiaohuan/svn

2：安装phpadmin 便于管理数据库

绿色包，解压，配置端口即可，包地址：http://pan.baidu.com/s/1qW7pNnY

修改下conf，我的是3606 默认3306端口。
